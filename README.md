## Project Description
This project applies Principal Component Analysis (PCA) for dimensionality reduction and DBSCAN for clustering.

The goal is to compare clustering performance before and after applying PCA.

---

## Dataset
- Iris dataset

---

## Methods Used
- Data preprocessing (cleaning and scaling)
- PCA (2 components)
- DBSCAN clustering
- Parameter tuning (eps, min_samples)

---

## Results
- PCA retained approximately 95.8% of the variance
- DBSCAN produced:
  - Original Data: 2 clusters, 8 noise points
  - PCA Data: 2 clusters, 5 noise points

---

## Conclusion
PCA improved clustering performance by reducing noise and simplifying the dataset structure.
